
## 📱 Tela de Login

Este projeto é uma aplicação móvel construída com a plataforma Expo, que oferece uma experiência de criar uma simples lista de tarefas, ao ser concluida, a tarefa pode ser excluida.

## 🖥️ Funcionalidades

- Cadastro de Tarefas
- Excluir Tarefas
